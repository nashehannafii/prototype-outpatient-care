# Heart Care Planner (Prototype)

This is a minimal Vite + React + TypeScript + Tailwind project wrapper for the `heart-care-planner.tsx` component.

Quick start:

1. Install dependencies

```bash
cd /Users/nashehannafii/MyProjects/prototype-new-project
npm install
```

2. Start dev server

```bash
npm run dev
```

Open the dev server URL (usually http://localhost:5173) in your browser.

Notes:
- The component uses Tailwind classes and `lucide-react` icons.
- If you want a production build: `npm run build` and `npm run preview` to preview locally.
